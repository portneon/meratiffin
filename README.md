# Mera Tiffin 🍱

A premium full-stack tiffin delivery service application.

## Features
- **Modern UI**: Built with React and Vanilla CSS (Glassmorphism design).
- **Full Stack**: Express.js backend with MongoDB integration.
- **Authentication**: Secure JWT-based auth with password hashing.
- **Order Management**: Subscription plans and order tracking.

## Tech Stack
- **Frontend**: React, Vite, Vanilla CSS.
- **Backend**: Node.js, Express, Mongoose.
- **Database**: MongoDB (Atlas).

## Setup
1. Clone the repo.
2. `cd backend` && `npm install`
3. `cd ../frontend` && `npm install`
4. Create `.env` in backend with `JWT_SECRET` and `MONGO_URI`.
5. Run `npm run dev` in both directories.
